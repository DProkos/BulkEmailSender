# Bulk Email Sender - Οδηγίες Χρήσης

## Τι είναι το Bulk Email Sender

Το Bulk Email Sender είναι μια desktop εφαρμογή για μαζική αποστολή εξατομικευμένων emails μέσω SMTP. Ιδανικό για:

- Μαζική αποστολή newsletters
- Ενημερωτικά emails σε πελάτες
- Προωθητικές καμπάνιες
- Εταιρική επικοινωνία

---

## Εγκατάσταση

### Απαιτήσεις
- Windows 10/11
- Python 3.10+ (για development)

### Εκτέλεση
1. Κατεβάστε το `BulkEmailSender.exe`
2. Διπλό κλικ για εκτέλεση
3. Δεν χρειάζεται εγκατάσταση!

## Βήμα-Βήμα Οδηγός

### 1️⃣ Ρύθμιση SMTP (Tab: SMTP Settings)

Συμπληρώστε τα στοιχεία του email server σας:

| Πεδίο | Περιγραφή | Παράδειγμα |
|-------|-----------|------------|
| Host | Διεύθυνση SMTP server | smtp.gmail.com |
| Port | Θύρα σύνδεσης | 587 (STARTTLS) ή 465 (SSL) |
| Username | Το email σας | user@example.com |
| Password | Κωδικός email | ******** |

**Συνήθεις ρυθμίσεις:**

**Gmail:**
- Host: `smtp.gmail.com`
- Port: `587`
- Encryption: STARTTLS
- ⚠️ Χρειάζεται App Password αν έχετε 2FA

**Outlook/Office 365:**
- Host: `smtp.office365.com`
- Port: `587`
- Encryption: STARTTLS

**Cosmote:**
- Host: `mailgate.cosmotemail.gr`
- Port: `587`
- Encryption: STARTTLS

Πατήστε **Test Connection** για να επιβεβαιώσετε τη σύνδεση.

---

### 2️⃣ Εισαγωγή Παραληπτών (Tab: Recipients)

#### Από Excel:
1. Πατήστε **Select Excel File**
2. Επιλέξτε αρχείο .xlsx
3. Αντιστοιχίστε τις στήλες:
   - **Email Column**: Στήλη με emails (υποχρεωτικό)
   - **Name Column**: Στήλη με ονόματα (προαιρετικό)
   - **Company Column**: Στήλη με εταιρείες (προαιρετικό)
4. Πατήστε **Load Recipients**

#### Χειροκίνητα:
- **Add Email Manually**: Προσθήκη ενός email
- **Add Multiple Emails**: Επικόλληση πολλών emails

#### Μορφή Excel:
```
Email              | Name        | Company
-------------------|-------------|------------
john@example.com   | John Doe    | Acme Corp
jane@example.com   | Jane Smith  | Tech Inc
```

---

### 3️⃣ Στοιχεία Εταιρείας (Tab: Company Settings)

Συμπληρώστε τα στοιχεία της εταιρείας σας για χρήση στα templates:

- Όνομα εταιρείας
- Διεύθυνση
- Τηλέφωνο
- Email
- Website
- Social Media links

Αυτά θα είναι διαθέσιμα ως μεταβλητές: `{{company.name}}`, `{{company.phone}}`, κλπ.

---

### 4️⃣ Δημιουργία Template (Tab: Template)

#### Subject (Θέμα):
Γράψτε το θέμα του email. Μπορείτε να χρησιμοποιήσετε μεταβλητές:
```
Καλημέρα {{name}}, νέα προσφορά για εσάς!
```

#### Email Body:
Πατήστε **📝 Edit Email Body...** για να ανοίξει ο επεξεργαστής.

**Διαθέσιμες μεταβλητές:**
- `{{name}}` - Όνομα παραλήπτη
- `{{email}}` - Email παραλήπτη
- `{{company}}` - Εταιρεία παραλήπτη
- `{{unsubscribe_link}}` - Link απεγγραφής

**Παράδειγμα HTML:**
```html
<p>Αγαπητέ/ή {{name}},</p>

<p>Σας ευχαριστούμε για το ενδιαφέρον σας.</p>

<p>Με εκτίμηση,<br>
{{company.name}}</p>

<p><small><a href="{{unsubscribe_link}}">Απεγγραφή</a></small></p>
```

#### Συνημμένα:
Πατήστε **Add Attachment** για να προσθέσετε αρχεία.

⚠️ Προσοχή: Μεγάλα συνημμένα (>10MB) μπορεί να απορριφθούν.

---

### 5️⃣ Αποστολή (Tab: Send)

#### Ρυθμίσεις:
- **Throttle Rate**: Καθυστέρηση μεταξύ emails (προτείνεται 2-5 sec)
- **Max Retries**: Επαναλήψεις σε περίπτωση σφάλματος

#### Κουμπιά:
- **▶️ Start Send**: Έναρξη αποστολής
- **⏸️ Pause**: Παύση
- **⏹️ Stop**: Διακοπή

#### Dry Run Mode:
Ενεργοποιήστε το για δοκιμή χωρίς πραγματική αποστολή.

## Συμβουλές

### Αποφυγή Spam Filters:
1. Χρησιμοποιήστε throttle rate τουλάχιστον 2 sec
2. Μην στέλνετε πάνω από 100 emails/ώρα
3. Προσθέστε unsubscribe link
4. Αποφύγετε ΚΕΦΑΛΑΙΑ και πολλά θαυμαστικά!!!

### Ασφάλεια:
- Οι κωδικοί αποθηκεύονται στο Windows Credential Manager
- Δεν αποθηκεύονται σε αρχεία
- Χρησιμοποιήστε App Passwords αντί για κανονικούς κωδικούς

### Crash Recovery:
Αν η εφαρμογή κλείσει απρόσμενα:
1. Ανοίξτε ξανά την εφαρμογή
2. Θα σας ρωτήσει αν θέλετε να συνεχίσετε
3. Τα ήδη σταλμένα emails δεν θα ξανασταλούν

## Αντιμετώπιση Προβλημάτων

### "Authentication Failed"
- Ελέγξτε username/password
- Για Gmail: Χρησιμοποιήστε App Password
- Ενεργοποιήστε "Less secure apps" (αν δεν έχετε 2FA)

### "Connection Timeout"
- Ελέγξτε τη σύνδεση internet
- Επιβεβαιώστε host και port
- Ελέγξτε το firewall

### "Rate Limit Exceeded"
- Αυξήστε το throttle rate
- Περιμένετε λίγο και δοκιμάστε ξανά
